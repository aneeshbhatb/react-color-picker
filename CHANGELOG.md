# Changelog

## 0.1.0-alpha.1 — 2026-05-31

### Added

- Alpha slider for selecting opacity.
- `classNames` prop to customize picker slots with CSS classes.
- `styles` prop to customize picker slots with inline styles.

## 0.1.0-alpha.0 — 2026-05-30

### Added

- Initial alpha release of `@aneeshbhat/react-color-picker`.
- Solid color picker with saturation/brightness and hue controls.
- Controlled usage via `value` and `onChange`.
